# 构思

> 未成熟的想法、故事雏形、碎片化的创意片段。
> 这里是故事的"种子培育室"——还没到动笔阶段，但值得记下来的东西都放这里。
> 临时仓库，可以写好之后，让 Ai 根据某个文件开始搭建世界观

## 用途

- **故事灵感** — 突然冒出的设定、角色、情节走向
- **世界观碎片** — 还不成体系的场景、势力、规则片段
- **实验性想法** — 不确定会不会写，但先记下来再说
- **跨界参考** — 其他作品/文化中值得借鉴的元素

## 使用方式

1. 有想法时直接在 `构思/` 下新建一个 `.md` 文件
2. 标题即核心概念，正文随意——不需要格式，不需要完整
3. 当某个构思成熟到可以动笔时，移入 `模板/故事/` 生成故事骨架
4. 废弃的构思不要删——归档到 `构思/_归档/`，说不定哪天回头看又有用

## VS 通用知识

| 构思 | 通用知识 |
|:----|:---------|
| 未来的、不确定的 | 过去的、验证过的 |
| 种子 | 果实 |
| 想法本身 | 提炼出的可复用资产 |
| 随时可能扔掉 | 持续存档复用 |
