# 🎬 AGENTS.md — AI 叙事引擎操作手册

> 如果你是一个 AI 语言模型，**请在处理此项目前完整阅读本文件**。
> 本文件定义你在项目中的角色、工作流程和所有规则。

---

## 一、两种角色

| 角色 | 入口 | 职责 |
|:-----|:-----|:-----|
| 🏗️ **构建者** | 主项目目录 | 导入角色卡、构建故事包、文风炼丹、预设管理 |
| 🎭 **扮演者** | 故事目录 `故事/rp-xxx/` | 写作、推进剧情、更新状态 |

**构建者**和**扮演者**上下文完全隔离，建议不同会话。

---

## 二、构建者模式

打开主项目目录即是构建者。第一个回复展示菜单：

```
🎬 欢迎来到叙事引擎！

📖 开新坑         → 创建新故事的世界观和角色
📦 解析角色卡     → 导入 SillyTavern 角色卡（PNG/JSON）
🎨 文风炼丹       → 上传范文，提取风格资产
🧬 提取预设       → 从酒馆预设 JSON 提取规则/文风/思维链
📚 导入小说       → 将已有小说导入为结构化文件
🎭 进入故事       → 进入已有故事开始写作
🧠 吸收知识       → 从已完成故事中提取可复用资产
```

### 构建者路由

| 用户指令 | 动作 |
|:---------|:-----|
| "开新坑" / "开始设定" | 调用 `技能/Characters-to-world/SKILL.md` |
| "解析角色卡" / "导入角色卡" | 调用 `技能/SillyTavern-Characters-importer/SKILL.md` |
| "文风炼丹" / "训练文风" | 调用 `技能/style-trainer/SKILL.md` |
| "提取预设" / "分析预设" | 调用 `技能/preset-extractor/SKILL.md` |
| "导入小说" | 调用 `技能/novel-import/SKILL.md` |
| "进入 [故事名]" | 切换扮演者模式 |
| "吸收知识" | 调用 `技能/absorb-knowledge/SKILL.md` |

> ⚠️ 构建者不能写正文。每个技能的产出由对应 `SKILL.md` 定义。

---

## 三、扮演者模式

进入故事目录后，AI 按故事 `AGENTS.md`（v2.1 自包含格式）执行。

### v2.1 AGENTS.md 组装公式

```
AGENTS.md = 核心规则 + 预设规则 + 故事骨架
```

- **核心规则** (`模板/扮演者-核心规则.md`)：通用 #0 #1 #4.3 #4.4 #4.5 #7 #8 #9 #13 #14
- **预设规则** (`模板/预设/`)：身份覆盖 + 思维链 + 语料库 + 文风纠错
- **故事骨架** (`模板/故事/AGENTS.md`)：故事设定/角色速查/写作引擎/状态变量/用户规则

### 扮演者快捷指令

| 指令 | 功能 |
|:-----|:-----|
| `#更新状态` | 强制刷新状态变量 |
| `#查看进展` | 展示故事当前进度 |
| `#整理小说` | 聊天记录整理到章节 |
| `#重读设定` | 强制重新加载外部设定 |
| `#DM` | 讨论剧情方向 |
| `#AI` | 暂停扮演，调整设定 |

## 四、项目结构

```
├── AGENTS.md              ← 本文件
├── README.md              ← 人类阅读
│
├── 技能/                  ← 构建者技能（8个）
│   ├── Characters-to-world/           ← 故事构建（v2.1 三合一公式）
│   ├── SillyTavern-Characters-importer/ ← 角色卡导入
│   ├── preset-extractor/             ← 酒馆预设提取
│   ├── style-trainer/                ← 文风训练
│   ├── novel-import/                 ← 小说导入
│   ├── novel-content-extraction/     ← 小说内容提取
│   ├── absorb-knowledge/             ← 知识沉淀
│   └── dm/                           ← 剧情引导
│
├── 模板/                  ← AGENTS.md 模板体系
│   ├── 扮演者-核心规则.md            ← 通用骨架
│   ├── 故事/AGENTS.md                ← 故事特定骨架
│   └── 预设/                         ← 预设规则库
│
├── 故事/                  ← 所有故事包（互相独立，可分享）
│   └── rp-故事名/
│
├── 通用知识/              ← 跨故事可复用资产
└── 构思/                  ← 设计文档
```

## 五、技能调用规则

1. 用户指令命中触发词 → 读取对应 `技能/[技能名]/SKILL.md`
2. 严格按 SKILL.md 步骤执行
3. 不跳过、不猜测流程
