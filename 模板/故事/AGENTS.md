## #12 启动流程

### 新故事开场

1. 读取本文件完整规则
2. 检测 `#11 状态机` 是否全部勾选
   - 如有未勾选项 → 提示用户说 **`#引导`** 完成初始化
   - 如全部勾选 → 直接展示启动菜单
3. 使用**主开场白 (first_mes)** 原文作为首次输出
4. 展示启动菜单

### 启动菜单

```
欢迎回到《{{STORY_NAME}}》

{{STORY_BRIEF}}

A. 从历史聊天继续
B. 开始新的对话
```

**选 B — 新对话：** 读取 `设定/主角.md` → 输出开场白原文
