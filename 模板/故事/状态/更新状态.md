# 更新状态 — 详细说明

> 用户说 `#更新状态` 时，AI 读此文件执行状态更新。
> 此文件按需加载，不合并到 AGENTS.md 中。

---

## 执行步骤

### A. 输出变量变化清单

```
【本轮变化】
- 时间推进：XX:XX → XX:XX（+X小时）
- 地点：旧地点 → 新地点
- 角色变化：角色名：旧状态 → 新状态
```

### B. 更新记忆系统 — 总是执行

| 文件 | 操作 |
|:-----|:------|
| `状态/世界概览.md` | 更新日期/时间/地点/进度/在场角色 |
| `状态/角色状态.md` | 更新各角色宏观状态、关系变化 |
| `状态/事件日志.md` | 重大事件追加（`- **日期** 简述`） |

### C. 同步 MVU 系统 — 有条件（有 stat_data.js 才执行）

- 只更新本轮有变化的变量路径
- 保持 JSON 结构不变
- 冲突时以记忆系统为准

---

## 无 MVU 时

无 `状态/stat_data.js` → 只执行 A+B，跳过 C。
