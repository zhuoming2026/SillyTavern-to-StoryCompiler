# 故事元数据

## 基本信息

| 字段 | 值 |
|:-----|:----|
| **创建日期** | `{{YYYY-MM-DD}}` |
| **故事状态** | `设定中` |
| **总章节数** | `0` |
| **最后会话** | `（空）` |
| **当前场景** | `未开始` |

### 状态说明

| 状态 | 含义 |
|:-----|:------|
| `设定中` | 世界观/角色/文风尚未完成 |
| `写作中` | 设定已完成，正文创作进行中 |
| `已完结` | 故事已完结 |
