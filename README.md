# 🎭 SillyTavern-to-StoryCompiler

> 👤 **打开项目后，跟 AI 说"读取 AGENTS.md"或"介绍下这个项目"。**

将 SillyTavern 角色卡（PNG/JSON）导入为结构化故事包，驱动 AI 沉浸式创作。

> ⚡ **适合你的就是最好的。**
> 这不是一个死板项目，是一个活的框架。
> **AI 是万能的**——流程不对告诉 AI 改，技能不满意让 AI 按你说的调。
> 所有改进记录到 `AGENTS.md`，项目自己进化。

---

## 🤷 说人话版

你有 SillyTavern 的角色卡（PNG/JSON），想用 AI 写故事。没有也没关系，可以现场构建。

三步走：
1. **解析角色卡** → 角色、世界观、开场白全提取
2. **选预设** → 从酒馆预设里提取文风、思维链、禁八股规则（可选）
3. **开始扮演** → AI 拿到完整手册，直接开写

两个人分活干：
- **构建者**（主项目）→ 跑腿的，解析角色卡、炼丹、搭世界观 [酒馆的写卡]
- **扮演者**（故事目录）→ 写手，只管写故事 [酒馆的玩卡]

**看不懂没关系。** 打开项目，跟 AI 说"介绍下"，它会告诉你。

```
下载客户端 → 打开项目 → 跟 AI 聊天 → 开始创作
```

### 1. 下载客户端

推荐 **Reasonix Desktop**：https://github.com/esengine/DeepSeek-Reasonix

不是必须。Hermes、任何能读 `AGENTS.md` 的 Agent 都行。


打开 Agent，工作目录指向刚 clone 的文件夹。

## 🧠 核心设计（v2.1）

### AGENTS.md 是怎么生成的？

```
AGENTS.md = 核心规则 + 预设规则 + 故事骨架

① 扮演者-核心规则.md     ← 通用骨架：身份、铁律、语料库、纠错、流程
② 扮演者-TGbreak-规则.md  ← 预设注入：文风、思维链、禁八股（可选）
③ 角色卡数据             ← 故事特定：角色、世界观、开场白提取的文风
```

### 预设系统

从酒馆预设 JSON 自动提取，生成可直接勾选的清单文件：

- 🎭 **角色灵魂** → 覆盖 AI 人格
- 🧠 **思维链** → COT-基础思维、深度瑟瑟、防变量出错
- 🎨 **文风** → 搞笑网文/基础/地道中文/轻小说/古风
- 📝 **纠错** → 禁AI口吻/杀八股/禁八股黑名单/比喻规范
- 📚 **语料库** → 瑟瑟多样化

## 🎮 Tips

**📖 不知道具体怎么用？** → 看 [USAGE.md](USAGE.md)，真实对话流程，从第一次打开项目到边玩边改身份加金手指。

**⚡ AI 是万能的。** 两个角色就是写卡和玩卡。把酒馆的世界书.json、预设.json 丢给 AI，直接让它给你炼。

**🎨 文风炼丹（⭐ 推荐）** — `style-trainer` 输入任意范文，AI 从多维度分析提炼专属文风。炼好存到 `通用知识/文风体系/`，开新坑直接套。

**📦 角色卡导入** — 支持 PNG/JSON（V2/V3），自动提取角色、世界书、开场白。

**🎭 故事包可分享** — `故事/rp-xxx/` 打包发给朋友，对方 AI 读 `AGENTS.md` 直接开玩。

**🧠 持续沉淀** — 玩完一个故事，"吸收知识"把有意思的设定存到 `通用知识/`，下次构建复用。
