# 🎭 SillyTavern-to-StoryCompiler

一个**文件驱动的 AI 叙事创作系统**——将 SillyTavern 角色卡导入为结构化文件，通过工作流引擎驱动 AI 进行创作。

```
下载客户端 → 打开项目 → 跟 AI 说 → 开始创作
```

---

## 🚀 快速开始

### 1. 下载客户端

推荐 **Reasonix Desktop** 桌面客户端：
👉 [https://github.com/esengine/DeepSeek-Reasonix](https://github.com/esengine/DeepSeek-Reasonix)

推荐桌面客户端。理论上来说，不是必须 Reasonix，你可以用 Hermes 等 Agent，让它自适应就好了。但我不推荐一些自带审核的 Agent 工具，DDDD。

### 2. 打开项目

```bash
git clone https://github.com/zhuoming2026/SillyTavern-to-StoryCompiler.git
```

打开 Reasonix Desktop → **工作目录选择** → 指向刚 clone 的文件夹 → 编辑 `reasonix.toml` 填入 API Key

### 3. 跟 AI 说...

| 你想做什么 | 跟 AI 说 |
|:-----------|:---------|
| 🆕 **了解系统** | "介绍一下这个项目如何运行" |
| ✍️ **开始写新故事** | "开始新故事" |
| 🌍 **构建世界观** | "开始设定" |
| 🎨 **文风炼丹（⭐）** | "文风炼丹" — 给范文，炼专属文风 |
| 📄 **解析角色卡** | 把 PNG 给 AI，说"解析这张角色卡" |
| 🧠 **保存创作资产** | "吸收知识" |
| 📦 **完结归档** | "归档" |

### 4. 🎮 游玩 Tips

> 这不是一个严格的项目，你不需要按部就班。**告诉 AI 你希望如何，它能自己改进。**

**💬 随便聊** — 导入角色卡后，你可以直接跟 AI **OOC**（跳出角色聊天），让它帮你改进角色、调整设定，甚至一起 brainstorm 剧情方向。

**🎨 文风炼丹（⭐ 强烈推荐）** — 使用 `style-trainer` 技能，输入任意格式的范文（小说原文/文风指南/SillyTavern JSON/提示词描述），AI 会从 8 个维度分析提炼出你的专属文风并输出单文件 `文风分析.md`。将炼好的文风保存到 `通用知识/文风体系/`，之后开新故事时直接应用、测试。玩得越多，文风越贴你的口味。

**🔞 你的 XP** — 不用藏着掖着，直接告诉 AI 你的偏好。它会记录到文风模型里，后续创作越来越贴合你的口味。

**📋 吃角色卡** — 把 SillyTavern 角色卡 PNG 丢进来，AI 提取角色设定、世界书、预设规则，你只需要选需要的部分。

**🧠 持续沉淀（⭐ 强烈推荐）** — 每次玩完一个故事，把其中有意思的设定（世界观、角色、体系规则等）提取到 `通用知识/` 保存。下次构建新世界观时，AI 可以直接复用这些资产，新故事的设定会既快又丰富。好文风、好设定，存了就一直在。

**🛠️ 项目自己进化** — 觉得缺什么功能？直接跟 AI 说"加一个能 xxx 的技能"，它能自己写新的技能模块。这个项目是活的。

> 💡 **相信 AI 是万能的，有什么要求就跟它说。**

## 系统架构

### 工作流

```
① 设定引导 → ② 文风确定 → ③ 开始写作 → ④ 修改 / ⑤ 讨论
                                              ↓
                                        ⑥ 知识沉淀
                                              ↓
                                        ⑦ 归档故事
```

### 技能模块

`技能/` 下的每个目录是一个独立能力模块：

| 技能 | 功能 |
|:-----|:------|
| `world-builder` | 引导用户逐步完善故事设定 |
| `style-trainer` | 从任意格式输入（小说/文风指南/JSON/口述）提取文风，输出单文件 `文风分析.md` |
| `dm` | 出剧情方向选项 |
| `novel-import` | 📖 完整导入小说（切章→LLM深度分析→10个结构化文件） |
| `novel-content-extraction` | 🔍 快速扫描小说（纯脚本秒出角色/章节/关键词） |
| `absorb-knowledge` | 从完成故事中提取可复用资产 |
| `archive-story` | 归档完成的故事并清空工作区 |
| `world-book-png` | 解析 SillyTavern 角色卡 PNG |

> 写作和修改由 AI 直接交互完成，无需独立技能模块。

## 目录结构

```
├── agent.md              ← 工作流控制器（灵魂文件，新会话先读）
├── 开始写作.md           ← 已合并到 agent.md（仅占位）
├── README.md             ← 本文件
├── reasonix.toml         ← AI 客户端配置
├── .gitignore
│
├── 配置/                 ← 全局配置
│   ├── config.json             ← 输出控制（字数/文风）
│   ├── user_profile.json       ← 用户身份
│   └── 扮演规则.md             ← 🆕 全局通用扮演规则（状态变量驱动）
│
├── 技能/                 ← 🛠️ 自动化能力模块（8个）
│
├── 模板/故事/            ← 新故事目录模板
│
├── 故事/                 ← 🔵 所有故事
│   └── rp-故事名/
│       ├── 设定/         ← 世界观 + 角色
│       ├── 风格/         ← 文风 + 范文 + 分析
│       ├── 配置/         ← 故事级配置 + 扮演规则（参照 `配置/扮演规则.md` 通用规则）
│       ├── 写作/         ← 小说章节 + 聊天记录
│       └── 状态/         ← 实时追踪（世界/角色/事件）
│
└── 通用知识/             ← 🧠 跨故事知识库（文风、设定、体系等，可复用资产）
```

### 🎬 两种创作模式

在 `agent.md` 中定义了两个模式，你可以随时切换：

| 模式 | 特点 | 适合谁 |
|:-----|:------|:-------|
| **🎬 导演模式**（默认） | 先出「规划」→ 你确认 → 再写正文 → 迭代修改 → 整理成章。每一步都有规划文档，精修细改。 | 想打磨文笔、控制剧情走向、追求质量的写作者 |
| **⚡ 冒险模式** | 直接推剧情，不经过规划步骤。AI 实时引入 NPC、给出方向选项，你选方向就继续写。 | 想快速体验剧情、喜欢惊喜和探索感的玩家 |

> 说 **"切换冒险模式"** 或 **"切换导演模式"** 即可随时切换，不需要重启会话。

## 🎬 使用演示

> 以下是一次完整游玩会话的真实流程，让你秒懂怎么跟 AI 配合。

——————

**👨：「给我介绍下项目」**

**🤖：** 这是一个文件驱动的 AI 叙事创作系统。您只需要跟我聊天说方向，我来完成设定构建、文风训练、章节写作、状态追踪等所有工作。当前故事目录为空，是否开始一个新故事？

**👨：「导入角色卡」**（把 PNG 拖进对话框，或给文件路径，如 `/Users/zzz/酒馆资料/催眠都市.png`）

**🤖：** 正在解析角色卡……提取完毕！

这是一个**现代都市世界**，你拥有催眠能力，可以通过眼神接触或特定指令对他人进行潜意识暗示。当前可用的主要角色已提取到 `设定/角色/`。

🌍 **世界观概览：** 2024 年，一座普通都市的地下，存在着少数觉醒催眠能力的个体。你扮演其中一员，刚刚发现自己的能力。

请问你需要修改设定吗？比如调整催眠的发动条件、能力限制等。

**👨：「降低催眠难度，可以直接发动」**

**🤖：** 好的，已修改：你现在可以无需任何前置条件直接发动催眠，只需目光接触即可瞬间生效。`设定/世界观/能力规则.md` 已更新。

这样改可以吗？如果可以，我们进入下一步——**文风训练**。

**👨：「ok」**

**🤖：** 现在我们开始确定文风。你有几种选择：
- 📖 **参考小说** — 给我几篇你喜欢的范文，我提炼文风
- 🎭 **使用角色卡自带文风** — 用刚才角色卡里自带的文风
- ✍️ **使用开场白文风** — 按开场白的风格来写
- 🆕 **自定义** — 直接描述你想要的风格

你想用哪种？

**👨：「参考这些小说，使用这个文风」**（附上范文）

**🤖：** 收到，正在运行 `style-trainer` 技能进行文风炼丹……已从 8 个维度（用词、句式、节奏、对话、描写、心理、氛围、叙事视角）完成分析，结果已保存到 `风格/文风分析.md`。

**👨：「开始吧，我选择导演模式」**

**🤖：** 好的，**导演模式**启动！每个场景我都会先出规划，等你确认后再写正文。

**👨：「开场白有哪些」**

**🤖：** 角色卡自带以下开场白：
1. 你在一家咖啡馆里，对面坐着你的第一个目标……
2. 深夜，你收到一条匿名短信：「他们发现你了。」
3. 你在心理咨询室结束了今天最后一个病人……

你想从哪个开始？

**👨：「规划第一章：了解这个系统，然后找个人试试」**

**🤖：** 📋 **场景规划：**
- **场地：** 城市商业区咖啡馆
- **角色：** 你 + 随机路人（测试能力）
- **侧重点：** 主角初次意识到自己的能力 → 犹豫 → 尝试 → 成功后的心理变化
- **氛围：** 日常中暗流涌动

请确认或修改规划，说「开始写」我就动笔 ✍️

——————

> 💡 **整个过程的共同点：** 你只需要给方向、给反馈、做决定。AI 负责所有执行层面的工作——解析角色卡、构建世界观、分析文风、写规划、写正文、更新状态。
