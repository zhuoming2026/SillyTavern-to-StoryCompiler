# 中文人名提取踩坑记录

## 问题概述

中文小说中提取人名比英文难得多——没有大写提示，名字和普通词混在一起。

## 已验证的错误模式

### 1. 动词/副词粘连

"陈箜说了" → 提取出 "陈箜说"（错）而非 "陈箜"（对）

**解法**：用零宽前瞻 `([\u4e00-\u9fff]{2,3})(?=[说|道|...])` 只匹配名字部分。

### 2. 副词插在名字和动词之间

"陈箜又说了" → 提取出 "陈箜又"（错）

**解法**：正则中间加可选副词 `([\u4e00-\u9fff]{2,3})(?:[又也就都才却]*)[说道...]`

### 3. 常用词被误判为人名

"什么" "不是" "不过" "心里" "忍不住" "妈妈" 等高频出现且满足 2-3 字条件。

**解法**：黑名单 + 首尾字过滤 + 独立出现验证。

### 4. 名字变体共存

"顾月凝" 和 "月凝" 是同一人。

**解法**：合并变体——按长度降序排列，短名是长名子串时合并计数，保留长名。

### 5. 全文出现 ≠ 是人名

"心里说" 中 "心里" 出现很多次但不是人名。

**解法**：独立出现验证——要求名字前后是标点/空格/常见分隔符，且 >= 3 次。

## 过滤层次（从粗到细）

```
Layer 1: 长度过滤     → 只要 2-3 字
Layer 2: 黑名单       → STOP_WORDS + 手工黑名单
Layer 3: 首字过滤     → 排除常见动词/代词开头
Layer 4: 尾字过滤     → 排除常见动词/助词/副词结尾
Layer 5: 独立验证     → 全文中以独立形式出现 >= 3 次
Layer 6: 变体合并     → 短名并入长名
```

## 黑名单分类

- **虚词/代词**：什么、怎么、不是、没有、她们、我们...
- **粗口/语气**：妈的、他妈、我草、我去...
- **动词短语**：忍不住、不由得、不知不觉...
- **称谓**：妈妈、爸爸、老师、老板、总裁...
- **方位词**：心里、眼中、面前、身边、背后...
- **专有名词片段**：陈箜说、陈箜笑...（特定小说的误匹配）
