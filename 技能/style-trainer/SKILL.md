# style-trainer — 文风与性癖提取器

## 职责

分析范文的写作方式（而非剧情内容），提取可复用的风格资产和性癖/XP 模式。

## 输入

```
故事/rp-故事名/风格/
├── 范文1.md         ← 必需（至少一篇）
└── 范文2.md         ← 可选（对照用）
```

## 输出

```
故事/rp-故事名/风格/风格分析/
├── 作者语言模型.md
├── 叙事节奏.md
├── 人物对白模型.md
├── 章节结构.md
├── 描写偏好.md
├── 风格模型.md    ← 含写作倾向分析
├── 禁忌清单.md
└── 风格摘要.md
```

以及：

```
故事/rp-故事名/写作/小说/测试章·文风验证.md
```

---

## 分析原则

**分析"怎么写"**：句长、用词、节奏、对白模式、描写偏好、性癖模式

**不要分析"写了什么"**：不总结剧情、不分析世界观、不记录角色关系

---

## 各文件定义

### 作者语言模型.md
- 句长统计：短句(≤15字) / 中句(16-30字) / 长句(>30字) 占比
- 用词风格：书面化程度、口语化、成语密度
- 语言习惯：喜欢连续动作描写、喜欢反问句、喜欢省略主语、喜欢段末留白

### 叙事节奏.md
- 对话 vs 叙述 的大致比例
- 节奏模型：识别常见结构
- 场景切换频率

### 人物对白模型.md
- 每个主要角色的说话特点
- 常见句式
- 引导语使用风格（有无引导语、动作代替引导语等）

### 章节结构.md
- 平均长度
- 开头方式（直接进入/环境开场/对话开场）
- 结尾方式（悬念/情绪停顿/信息揭露）

### 描写偏好.md
- 四大类占比：动作 / 心理 / 环境 / 设定
- 作者重点描写什么、忽略什么

### 风格模型.md（含写作倾向分析）

若范文含成人内容，分析以下维度：

**推进模式**：慢热 / 快速进入 / 强剧情驱动

**描写重点**：动作 / 感官 / 心理 / 权力关系

**镜头距离**：近景为主 / 中景 / 远景

**情绪结构**：压抑 → 试探 → 失控 → 释放

**XP/性癖分析**（新增）：
- 叙事节奏倾向（快节奏/细腻铺陈/对话驱动/描写驱动）
- 权力动态（支配/服从/平等/对抗）
- 敏感带偏好（哪些身体部位被重点描写）
- 感官描写倾向（视觉/听觉/触觉/环境 的侧重）
- 声音描写（喘息/叫床/对话/沉默 的比例）
- 节奏偏好（快速冲刺 / 缓慢研磨 / 有节奏变化）
- 情感基调（温暖/沉郁/轻松/紧张/悲壮）
- 特殊癖好标签（如有：足/束缚/放置/露出/其他）

**⚠️ 禁止**：记录具体情节。只记录模式。

### 禁忌清单.md
作者极少使用的内容（很少环境描写、很少比喻等）

### 风格摘要.md
一页速查，含核心原则 5-8 条

## 测试章生成

8 份分析资产生成后，生成测试章到 `故事/rp-故事名/写作/小说/测试章·文风验证.md`：

**规范**：
- 500-800 字，独立日常场景（无需承接剧情）
- 严格遵循所有已提取的风格规律
- 每处写法用 `【】` 标注依据文件
- 末尾附验证清单

**验证清单**：
```markdown
| 检查项 | 是否符合？ |
|--------|-----------|
| 句长分布与范文一致 | ☐ 是 ☐ 否 |
| 对话比例接近原文 | ☐ 是 ☐ 否 |
| 角色说话方式符合 | ☐ 是 ☐ 否 |
| 描写偏好保持一致 | ☐ 是 ☐ 否 |
| 风格符合预期 | ☐ 是 ☐ 否 |
| 未触碰禁忌清单 | ☐ 是 ☐ 否 |
```

---

## 输出要求

- 只输出风格规律
- 不要输出剧情总结
- 不要输出世界观总结
- 不要输出角色关系总结
- **让另一个 AI 仅读这些文件就能模仿原作者续写**
