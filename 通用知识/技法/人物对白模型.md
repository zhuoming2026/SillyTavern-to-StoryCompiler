# 人物对白模型

> 综合四篇范文提取，视角修正为NTL（主角=主人/支配者/征服者）
> 女性角色定位：被征服者/母狗/奴隶/性奴

---

## 一、主角（征服者/主人）的说话特点

### 1.1 核心基调

**命令式 + 羞辱式 + 占有式**

主角（作为支配者）的语言风格统一为：**简短、有力、充满占有欲和羞辱性**。极少哀求、极少犹豫、极少询问意见。

### 1.2 常见句式

| 句式类型 | 示例 | 使用场景 |
|----------|------|----------|
| **命令句（祈使）** | "跪下。""张开嘴。""趴好。""叫主人。" | 任何权力展示时刻 |
| **反问羞辱** | "骚货，你这奶子是不是更大了？""就这么欠肏？" | 调情/前戏 |
| **占有宣示** | "你是我的母狗。""记住谁是你的主人。" | 确立权力关系 |
| **评价性点评** | "真紧……""你这骚屄夹得真舒服。" | 性交过程中 |
| **引导性提问** | "舒不舒服？""想不想要？""谁是你主人？" | 调教/确认臣服 |

### 1.3 语气程度

| 范文 | 语气特点 |
|------|----------|
| 范文1 | 粗口浓度最高，"肏死你！骚妈妈！"类型 |
| 范文2（NTL修正） | 带优越感的征服者口吻，"乖乖听话，张开腿" |
| 范文3 | 相对克制，更多是诱导性语言，"姐姐在叫你呢" |
| 范文4 | 既有调教式温柔又有暴虐命令，"乖，把奶子捧好" |

### 1.4 引导语使用

**风格：动作代替引导语 > 标准引导语 > 无引导语**

```
标准：      "张开腿。"他冷冷地命令道。
↑ 最常见：  他一把抓住她的头发往下按，"给老子舔干净。"
无引导语：  "骚货。""母狗。""叫主人。"
```

## 二、女性角色（被征服者/母狗/奴隶）的说话特点

### 2.1 核心基调

**羞涩 → 抗拒 → 呻吟 → 臣服 → 自我羞辱**

女性角色的对话是一条递进的堕落曲线，随着被征服的程度加深，语言从"拒绝"变为"渴望"。

### 2.2 常见句式

| 阶段 | 句式类型 | 示例 |
|------|----------|------|
| **抗拒期** | 微弱拒绝/道德挣扎 | "不行……我们是……母子/师徒……""别……这样……" |
| **动摇期** | 半推半就/自我开脱 | "就……就这一次……""只含一下……好么……" |
| **沉沦期** | 呻吟/请求/渴望 | "还要……再深一点……""好胀……好满……" |
| **臣服期** | 自称化/主动羞辱 | "我是主人的母狗……""骚屄想吃大鸡巴……" |
| **彻底奴隶化** | 自我贬低/哀求 | "把妈妈干成只会产奶的母猪吧……""求主人灌满……" |

### 2.3 常用称呼体系

**男性主角对女性称呼（羞辱/宠物化）：**
```
骚货、母狗、骚妈妈、母狗、母畜、肉便器、小浪蹄子、奶牛、奴隶
```

**女性对主角称呼（尊崇/臣服）：**
```
主人、老公、好儿子、少爷、主人大人、老公大人、爸爸（特殊情境）
```

### 2.4 语气词偏好

| 语气词 | 使用者 | 功能 |
|--------|--------|------|
| 呢/嘛/哟 | 女性（调情/挑逗/诱导时） | "姐姐在叫你呢"（范文3） |
| 啊/哦/嗯 | 女性（呻吟时） | "啊……好深……嗯……" |
| 吧 | 女性（半推半就时） | "就含一下……好么……" |
| 了/啦 | 女性（高潮时） | "去了……去了啦……" |

## 三、对话与动作的比例

| 场景 | 对话占比 | 规律 |
|------|----------|------|
| 调情/前戏 | ~50% | 对话与身体接触交替进行 |
| 性交中 | ~30% | 动作为主，对话辅助（命令+浪叫） |
| 高潮 | ~15% | 主要是无意义的呻吟/感叹词 |
| 余韵 | ~40% | 对话恢复，通常伴随羞辱性对话 |

## 四、对白中的权力标记

NTL视角下，对话中的权力关系必须通过以下方式明确标记：

### 4.1 空间位置标记
```
主角站着 ↔ 女性跪着/趴着
主角俯视 ↔ 女性仰视
主角不动 ↔ 女性主动服务
```

### 4.2 行动权标记
```
主角下达行动的指令  ↔  女性执行指令
主角获得快感        ↔  女性提供快感
主角决定节奏/深度   ↔  女性接受/承受
```

### 4.3 语言标记
```
主角使用命令句/反问句  ↔  女性使用请求句/回应句
主角称呼"你"          ↔  女性自称"母狗/奴隶"
主角少说话多动作      ↔  女性多浪叫多回应
```

## 五、对白模型速查表

| 要素 | 主角（主人） | 女性（奴隶） |
|------|-------------|-------------|
| **句类** | 命令句、反问句、陈述句 | 请求句、回应句、感叹句 |
| **语气** | 坚定、冷淡、羞辱、亢奋 | 羞涩、颤抖、呻吟、狂热 |
| **词汇** | 脏话、器官名词、命令动词 | 自称词、哀求词、呻吟词 |
| **句长** | 短句为主 | 短句+省略号 |
| **称呼自己** | 我/老子/主人 | 我（初期）→ 母狗/奴隶（沉沦后） |
| **称呼对方** | 骚货/母狗/名字 | 主人/老公/少爷 |
| **引导语** | 动作代替为主 | 动作配合呻吟 |
