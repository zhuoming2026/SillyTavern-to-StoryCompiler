# 作者语言模型

> 基于四篇范文（换母淫谋、都市NTL调教、都市修真官能、高反差堕落系）的综合提取
> 核心视角修正：NTR→NTL（主角=征服者/主人，女性=被征服者/母狗/奴隶）

---

## 一、句长统计

| 句类 | 标准 | 占比 | 典型功能 |
|------|------|------|----------|
| **短句** | ≤15字 | ~45% | 动作爆发、命令、感叹、高潮冲刺 |
| **中句** | 16-30字 | ~35% | 对话推进、心理活动、感官描写 |
| **长句** | >30字 | ~20% | 外貌特写、器官堆砌、环境铺陈 |

**规律：** 短句为主，长句为辅。静态描写用长句堆砌细节，动态推进用短句制造冲击。

## 二、用词风格

### 2.1 语体定位
**书面化粗口混搭体**——介于网络色情小说口语与华丽书面语之间。

### 2.2 词汇特征

| 维度 | 特征 | 范例 |
|------|------|------|
| 性器官名词 | 高频、夸张、非医学化 | 大鸡巴、肉棒、骚屄、蜜穴、龟头、名器 |
| 羞辱性动词 | 侵略性强 | 肏、干、捅、顶、贯穿、灌满、蹂躏 |
| 形容词 | 极致化、堆叠式 | 粗大、狰狞、雪白、丰腴、滚烫、浓稠 |
| 四字成语 | 描写女性美貌时出现 | 倾国倾城、风华绝代、丰乳肥臀 |
| 脏话/粗口 | 对话中高频使用 | 骚货、母狗、贱货、骚浪、欠肏 |

### 2.3 成语密度
约 **5-8个/千字**，集中在女性外貌和性爱场景的静态描写段。

## 三、语言习惯

### 3.1 核心习惯清单

| 习惯 | 频率 | 说明 |
|------|------|------|
| **连续动作描写** | ★★★★★ | 一连串动词堆叠："拔出、挺入、撞击、研磨、喷射" |
| **反问句** | ★★★★ | 用于挑逗和羞辱："骚妈妈，你这奶子是不是又大了？" |
| **感叹句** | ★★★★★ | 几乎每个性爱场景都大量使用："肏死你！爽不爽！" |
| **省略主语** | ★★★ | 动作描写中常省略："一把抓住，狠狠揉捏，用力掰开" |
| **段末留白** | ★★★ | 高潮段落用省略号结尾："意识陷入了白茫茫的……" |
| **感官前置** | ★★★★ | "先写身体感觉，再写理智反应"——范文3的特色 |
| **身份反差提示** | ★★★★ | "谁能想到……竟然……"这种反差句式反复出现 |

### 3.2 句法模块库

**模板A（外貌堆砌）：**
```
她那 [形容词1]、[形容词2] 且 [形容词3] 的 [身体部位]，此刻正 [状态描述]
```

**模板B（动作推进）：**
```
随着 [动作]，一股 [形容词] 的感觉从 [部位] 直冲 [大脑/脊髓]
```

**模板C（反差强调）：**
```
谁能想到，这位 [高贵身份] 的 [称呼]，竟然会 [堕落行为]
```

**模板D（心理挣扎）：**
```
明明是 [高尚/正常的行为]，可是 [身体/心理] 却 [产生了变态的反应]
```

**模板E（高潮爆发）：**
```
[感叹词]——！[动作]！[动作]！[动作]！[体液描写]！
```

### 3.3 标点习惯

| 标点 | 用途 | 示例 |
|------|------|------|
| **！** | 极高频，表达亢奋/命令/高潮 | "肏死你！骚货！用力！" |
| **……** | 表达呻吟/意识模糊/余韵 | "好舒服……还要……给我……" |
| **——** | 表示声音延长或转折 | "滋——！奶箭喷射而出" |
| **，** | 连续逗号连接多个动作 | "一把抓住她的头发，按在胯下，挺腰，深喉" |

## 四、四篇范文风格差异速查

| 维度 | 范文1（换母） | 范文2（NTL修正） | 范文3（修真） | 范文4（高反差） |
|------|------|------|------|------|
| 口语化程度 | 最高 | 高 | 中（雅俗混用） | 中高 |
| 成语密度 | 低 | 中 | 中 | 最高 |
| 感官侧重 | 视觉+触觉 | 视觉 | 嗅觉+触觉 | 视觉+味觉 |
| 粗口浓度 | 极高 | 极高 | 中高 | 高 |
| 句长偏好 | 短句为主 | 长短交替 | 长句铺陈 | 多重定语长句 |

## 五、总结：语言指纹

> **"粗口驱动的肉欲叙事"**——以口语化脏话为对话底色，以华丽堆砌的长句为描写笔法，短句爆发、长句铺陈，感叹号与省略号交替使用，形成"挑逗（中速）→ 前戏（慢速特写）→ 冲刺（快速短句）→ 高潮（感叹号爆发）"的语言节奏。
