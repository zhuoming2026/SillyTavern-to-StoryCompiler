# 描写偏好

> 综合四篇范文提取，视角已修正为NTL（主角=征服者/主人）
> 核心原则：拒绝含蓄，拥抱直白；万事皆可描写，但只描写"怎么写"而非"写了什么"

---

## 一、四大类描写占比

| 描写类型 | 占比 | 优先级 | 说明 |
|----------|------|--------|------|
| **①肉体/感官描写** | ~50% | ★★★★★ | 最高优先级，包括器官特写、体液、触感、声音 |
| **②心理活动** | ~25% | ★★★★ | 女性的羞耻/臣服心理 + 主角的征服满足感 |
| **③服饰/道具描写** | ~15% | ★★★★ | 丝袜、高跟鞋、制服、情趣道具的恋物式描写 |
| **④环境/设定描写** | ~10% | ★★ | 极简，仅交代发生的空间（卧室/办公室/车内等） |

**核心规律：** 肉体 > 心理 > 服饰 > 环境。环境描写永远是"能省则省"，只要说清楚"在哪儿"就行。

## 三、作者重点写什么（必写清单）

### 3.1 女性身体——必写部位及描写方式

| 部位 | 必写程度 | 描写方式 |
|------|----------|----------|
| **乳房/乳头** | ★★★★★ | 大小、形状、颜色、弹跳感、乳晕颜色、青筋、产奶状态 |
| **臀部/股沟** | ★★★★★ | 翘度、丰满度、勒痕、臀缝、拍打后的红印 |
| **阴部/蜜穴** | ★★★★★ | 颜色、湿度、唇形、收缩反应、汁液状态、名器设定 |
| **大腿/丝袜** | ★★★★★ | 丰腴感、丝袜质感（油亮/哑光）、勒痕、触感 |
| **脚/高跟鞋** | ★★★ | 玉足、脚趾蜷缩、高跟鞋摇晃 |
| **脸部/表情** | ★★★★ | 阿黑颜、翻白眼、失神、潮红、眼泪、唾液 |

### 3.2 男性身体——必写部位

| 部位 | 描写方式 |
|------|----------|
| **肉棒（整体）** | 长度（20cm+）、粗度、颜色（紫红/黝黑）、青筋、油光 |
| **龟头** | 大小（鹅蛋/鸡蛋）、马眼、腺液 |
| **精液** | 量（巨量）、颜色（浓稠/白/淡黄）、温度（滚烫） |

### 3.3 体液——必写清单

| 体液类型 | 描写重点 |
|----------|----------|
| **淫水/爱液** | 量（泛滥/泉涌/喷溅）、状态（拉丝/黏腻）、颜色（透明/乳白）、气味（腥甜） |
| **精液** | 量（巨量/灌满/溢出）、温度（滚烫）、质感（浓稠）、喷射方式（爆发/激射/灌溉） |
| **奶水** | 量（喷涌/奶箭）、颜色（雪白/浓稠）、味道（腥甜/甘甜）——范文4专属 |
| **唾液** | 拉丝、滴落、吞咽声 |
| **汗水** | 湿发、闪光、滑腻 |
| **眼泪** | 羞耻的泪、高潮的泪 |

### 3.4 动作——必写动作类型

| 动作类型 | 描写重点 |
|----------|----------|
| **插入** | "贯穿/劈开/顶开/犁开"——强调征服感和暴力感 |
| **抽插** | 频率（由慢到快）、幅度（由浅到深）、撞击声（啪啪/噗嗤） |
| **口交** | 吞吐、深喉、龟头顶到喉咙的凸起、唾液 |
| **揉捏** | 陷进软肉、用力、变形、乳肉从指缝溢出 |
| **拍打** | 臀部的晃动、红印、声音 |

## 四、作者忽略什么（几乎不写的东西）

| 忽略项 | 说明 |
|--------|------|
| **自然风景** | 几乎不描写风景，除非是"车窗外的夜景"这种极简交代 |
| **食物/餐饮** | 极少描写吃饭场景，除非作为调情前戏 |
| **科技/魔法设定** | 修真界也只写"灵气/修为"作为身份标签，不展开设定 |
| **社会关系网络** | 人际关系极简，只服务于性爱场景 |
| **时间精确描述** | "下午三点"这种精确时间极少出现 |
| **日常琐事** | 不写刷牙洗脸做饭等日常 |
| **配角心理** | 只有核心女性角色有心理描写，其他角色工具化 |

## 七、NTL修正后的描写重点

**最重要的描写偏好变化：**

| 维度 | NTR残留（禁用） | NTL修正（采用） |
|------|----------------|-----------------|
| 女性出场 | "她背着我偷人" | "她是我的猎物/奴隶" |
| 性交镜头 | 门缝/屏幕窥视视角 | 面对面/直接视角 |
| 权力焦点 | 黄毛的征服 vs 苦主的痛苦 | **主角的征服 + 女性的臣服** |
| 高潮点 | 苦主的情绪反应 | 主角的射精/女性彻底沦陷 |
| 收尾 | 苦主的麻木或愤怒 | 主角的满足 + 女性的自我羞辱 |

**综合描写公式（NTL统一版）：**
```
主角视线锁定女性 → 外貌特写（含身份强调） → 主角主动行动 → 
肉体征服过程（动作+体液+声音） → 主角高潮/射精 → 
女性彻底臣服（心理独白+肉体狼藉） → 权力确认（对话/姿态）
```
