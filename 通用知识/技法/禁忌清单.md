## 一、描写禁区（绝对避免）

| 编号 | 禁忌内容 | 原因 |
|------|----------|------|
| **1** | **唯美/文艺/含蓄的性描写** | 拒绝任何"红烛昏罗帐"式的诗意表达，必须直白粗俗 |
| **2** | **过于书面化/学术化的性器官名词** | 拒绝"阴茎""阴道""阴阜""会阴"等医学术语，必须用俗语 |
| **3** | **自然风景/环境氛围的长篇描写** | 风景描写几乎为零，空间只做极简交代 |
| **4** | **日常琐事/非性场景的冗长描写** | 不写做饭、上班、开会等日常（除非作为性爱前戏的铺垫） |
| **5** | **复杂的世界观/魔法/科技设定** | 设定极简，只作为身份标签存在 |
| **6** | **纯粹的精神恋爱/情感交流** | 所有情感互动必须与性欲/权力相关 |
| **7** | **男性角色的示弱/哀求/软弱** | 主角永远是支配者，不能露出软弱一面 |
| **8** | **女性角色的长期反抗/复仇** | 女性可以被短暂抗拒，但最终必须臣服 |
| **9** | **势均力敌的平等性爱** | 性爱必须有明确的权力落差（男支配女臣服） |
| **10** | **慢节奏的纯日常过渡** | 日常场景必须快速过渡到性相关的内容 |

## 三、语气/情绪禁区

| 编号 | 禁忌内容 | 原因 |
|------|----------|------|
| **1** | **过度悲伤/抑郁** | 情绪基调是亢奋、燥热、背德刺激，而非悲伤 |
| **2** | **道德批判** | 叙事不带道德批判立场，只有感官刺激 |
