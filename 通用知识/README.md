# 通用知识库

> 跨故事可复用的创作资产。
> 每个文件都是从已归档或已完结故事中提取的、超越具体故事的有用内容。
> 总之你让 Ai 提取就完事了

---

## 目录结构

```
通用知识/
├── 写作技法/          ← 怎么写——叙事节奏、描写偏好、对白模型、风格参考
├── 文风体系/          ← 什么风格——范文合集、风格融合指南
├── 体系设定/          ← 怎么设定——能力体系、世界规则、核心设定
├── 世界设定/          ← 什么世界——各类型世界观背景和角色体系
├── 调教技巧/          ← 角色关系与情感张力构建的工具参考
└── 规划方法论/        ← 怎么规划——创作总结、脉络说明、创意池
```

## 各文件夹说明

### 写作技法
通用的写作方法论，不依赖具体故事背景：
- **叙事节奏** — 起承转合、高潮分布、篇幅控制
- **描写偏好** — 中景原则、感官层次、边界变化写法
- **人物对白模型** — 口语化、角色差异、对白功能
- **风格参考** — 不同叙事风格的框架参考
- **禁忌清单 / 风格摘要** — 快速检索的写作红线与偏好总结

### 文风体系
各种风格的实际范文和融合指南：
- **范文** — 古风、都市、仙侠、悬疑等不同类型的完整段落
- **风格融合指南** — 如何将多种风格融合成统一叙事
- **叙事风格参考** — 不同叙事方向的框架说明

### 体系设定
可跨故事复用的机制模板：
- **能力体系** — 修炼体系、超能体系等
- **系统机制** — 数值框架、成长路径
- **核心设定** — 势力分布、世界规则

### 世界设定
各类型世界的背景参考，作为世界观构建的素材：
- 现实世界·超能、架空奇幻、都市等不同类型的世界背景设定
- 角色设计参考模板

### 调教技巧
角色关系与情感张力构建的工具参考：
- **角色关系发展指南**
- **情感张力构建技巧**
- **角色互动写作规则**

### 规划方法论
大型创作项目的规划思路：
- **创作总结** — 各类已完结项目的复盘
- **创意池** — 碎片化灵感积累
- **故事脉络** — 原故事的整体脉络和关键节点

## 使用方式

1. 新故事需要某个方向的内容时，先去对应文件夹找参考
2. 已有的可以直接引用或改编
3. 创作过程中发现可复用的新内容，归档到对应文件夹
4. **每个文件夹至少保留一个范文**，避免文件夹清空后丢失参考方向

## VS 构思

| 通用知识 | 构思 |
|:---------|:-----|
| 过去的、验证过的 | 未来的、不确定的 |
| 果实 | 种子 |
| 提炼出的可复用资产 | 想法本身 |
| 持续存档复用 | 随时可能扔掉或发展 |
